# Bob's Big Adventure

シューティングゲーム - Python/Pygame製

## 概要

「Bob's Big Adventure」は、6つのユニークなボスと戦うシューティングゲームです。各ボスを倒すと新しい装備がアンロックされ、ゲームプレイの幅が広がります。装備なしでクリアすると・・・？

## 必要環境

- **Python 3.7以上**（Python 3.12推奨）
- **必要なライブラリ**（自動インストール対応）:
  - pygame >= 2.5.0
  - pyttsx3 >= 2.90

## インストール方法

### 方法1: 自動インストール（推奨）

```bash
# ゲームを起動するだけで必要なライブラリが自動的にインストールされます
python shooting_game.py
```

### 方法2: 手動インストール

```bash
# 依存ライブラリを手動でインストール
pip install -r requirements.txt

# ゲームを起動
python shooting_game.py
```

## 操作方法

### 基本操作
- **矢印キー**: 移動
- **Z**: 射撃
- **ESC**: ポーズ

### メニュー操作
- **↑↓**: 選択
- **Enter**: 決定
- **E**: 装備画面
- **S**: セーブ
- **L**: ロード

- **V**: 弾の種類を切り替え（ノーマル/ホーミング/スプレッド）

### ダッシュ
- **← ←** (2回押し): 左にダッシュ

## ゲームシステム

### 装備システム

ボスを倒すと、新しい装備がアンロックされます。

装備画面（Eキー）で各装備のON/OFFを切り替えられます。

<details>
<summary>⚠️ 装備の詳細（ネタバレ注意）</summary>

- **ホーミング弾**: 自動追尾する弾丸
- **リーフシールド**: 周囲を回転する防御壁
- **拡散弾**: 3方向同時発射
- **ダッシュ**: 緊急回避・無敵時間付き
- **HPブースト**: 最大ライフ増加

</details>

### 星システム

レベルクリア時に星が獲得できます。クリア条件によって星の色が変わります。
全ての星を集めることが真のエンディングへの鍵？

### セーブ機能

- **2つのセーブスロット**: 異なる進行状況を保存可能
- **手動セーブ**: レベルクリア時にセーブ可能
- **復元機能**: 装備状態・クリア状況を完全復元

## 特殊要素

### カウントダウンシステム
各ボス戦開始前にカウントダウンが表示されます。音声合成による演出付き。

## トラブルシューティング

### 音声が再生されない
pyttsx3が正しくインストールされているか確認してください：
```bash
pip install --upgrade pyttsx3
```

### BGMが再生されない
`assets/audio/`フォルダに音楽ファイルが存在するか確認してください。

### セーブデータが読み込めない
`saves/`フォルダの権限を確認してください。必要に応じて削除して新規プレイしてください。
