==================================================
     Bob's Big Adventure - クロスプラットフォーム版
==================================================

このバージョンは Windows / Linux / Mac 全てで動作します！


【必要なもの】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Python 3.8 以上

まだインストールしていない場合:
→ https://www.python.org/downloads/
  からダウンロードしてインストール


【遊び方（超簡単！）】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ Windows の場合:
  1. setup_and_play.py をダブルクリック
     または
     コマンドプロンプトで: python setup_and_play.py

■ Linux / Mac の場合:
  ターミナルで: python3 setup_and_play.py

初回起動時に必要なライブラリが自動インストールされます！
2回目以降は即座にゲームが起動します。


【操作方法】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▼ タイトル画面
  スペースキー - ゲーム開始 / 進行
  T キー - タイトル画面に戻る
  ESC キー - ゲーム終了

▼ ステージ選択
  ↑↓ キー - ステージ選択
  スペースキー - 決定
  ESC / T キー - タイトルに戻る

▼ バトル画面
  矢印キー - プレイヤー移動
  スペースキー - 弾を発射
  P キー - ポーズ
  T キー - タイトルに戻る（ポーズ中）

▼ 装備画面
  ↑↓ キー - アイテム選択
  スペースキー - 装備ON/OFF
  ESC / T キー - タイトルに戻る


【ゲーム目標】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✦ 全6ステージのボスを倒してゲームクリア！
✦ ステージクリア後、便利な装備アイテムを獲得

【セーブデータ】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

【トラブルシューティング】
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: "Python が見つかりません" と表示される
A: Python をインストールしてください
   → https://www.python.org/downloads/
   インストール時に "Add Python to PATH" に必ずチェック！

Q: pip が動かない
A: コマンドで手動インストール:
   Windows: python -m pip install pygame numpy
   Linux/Mac: python3 -m pip install pygame numpy

Q: 画面が真っ暗 / 音が出ない
A: - グラフィックドライバを最新版に更新
   - システムの音量設定を確認
   - assetsフォルダが同じ場所にあるか確認

Q: "ModuleNotFoundError" エラー
A: setup_and_play.py を使って起動してください
   直接 shooting_game.py を実行すると依存関係が解決されません
